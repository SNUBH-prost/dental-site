# 치과 재료/술식 문서 작성

$ARGUMENTS 에 대해 아래 사이트 마크다운 형식으로 완성된 문서를 작성하세요.
인수가 없으면 먼저 어떤 재료/술식인지 물어보세요.

---

## 사이트 지원 마크다운 요소 (정확히 이 형식으로만 사용)

### 단계 배지
```
<kbd>Phase 1</kbd> **단계 제목 (영문병기)**
```

### 타이밍 블록 (각 Phase 시작부에)
```
<dl>
  <dt>정확한 시간</dt><dd>00:00 ~ 00:30 (총 30초)</dd>
</dl>
```

### 항목 목록 (물성·비교표 등)
```
<dl>
  <dt>항목명</dt><dd>값 또는 설명</dd>
  <dt>항목명</dt><dd>값 또는 설명</dd>
</dl>
```

### 강조 박스
```
<div class="tip"><b>💡 보철과 디테일</b>내용 — 경험적 팁, 임상 포인트</div>

<div class="warning"><b>⚠️ 주의</b>내용 — 실수하기 쉬운 포인트, 기전 설명 포함</div>

<div class="danger"><b>🚫 역효과 주의</b>내용 — 잘못 하면 생기는 합병증/실패</div>

<div class="note"><b>ℹ️ 안내</b>내용 — 배경 지식, 원리 설명</div>
```

### 접이식 보충 설명
```
<details>
<summary>🔬 재료 원리 / 배경지식</summary>

내용

</details>
```

### 인라인 강조
- 수치 하이라이트: `<mark style="background:#fef08a">10,000 RPM</mark>`
- 키워드 밑줄: `<u>지르코니아 전용 다이아몬드 버</u>`
- 굵게: `**내용**`

## 문서 구조 (이 순서로 작성)

```
## 📋 단계별 표준 술식 매뉴얼

---

(재료·술식 개요가 필요하면 ## 개요 섹션을 먼저)

<kbd>Phase 1</kbd> **단계명 (영문)**

<dl>
  <dt>정확한 시간</dt><dd>00:00 ~ 01:00</dd>
</dl>

1. 구체적인 step — 수치 포함, <u>핵심 재료명</u> 밑줄 표시
2. ...

<div class="warning"><b>⚠️ 주의</b>이 단계에서 가장 많이 틀리는 포인트 + 기전 설명</div>

---

<kbd>Phase 2</kbd> ...

<div class="tip"><b>💡 보철과 디테일</b>경험적 팁</div>

---

(필요 시)
<details>
<summary>🔬 재료 원리</summary>
배경 지식
</details>
```

---

## 작성 원칙

- **수치는 반드시 포함**: RPM, mm, 시간, 비율, 온도 등 — `<mark>` 로 하이라이트
- **핵심 재료/도구명**: `<u>` 밑줄 표시
- **각 Phase마다 타이밍 블록** `<dl>` 포함 (시간이 중요한 재료는 특히)
- **Warning·Tip은 각 Phase 끝에 배치** — 단순 주의 나열 금지, 왜 그런지 기전/이유 설명 포함
- **번역투 금지** — 교수가 전공의에게 설명하듯 자연스러운 한국어, 전문 용어 영문 병기
- **바로 붙여넣기 가능한 완성본** 출력 — 추가 편집 최소화
